### <a name="工程介绍">工程介绍</a>
此工程用来承载商标后台部分业务，技术栈为React，使用TypeScript来约束类型定义。工程内的部分文件描述如下：

* app

  - common

    公共模块，详见README

  - demo

    业务结构示例

  - 其它业务模块

    一个业务一个文件夹

* build

  包含打包业务模块的方法

* dest

  `dev`模式下，业务模块被打包在这里，可用来代理线上文件进行调试，或进行本地开发

* template

  - index.ejs

    各业务的标准模板，打包时，会将js、css的路径，以一定的规律传入此模板，随后编译为html，供用户访问

* vendor

  多个业务会用到的依赖，统一放在这里
